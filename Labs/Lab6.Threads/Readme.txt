Вариант 22
Создать приложение выполняющее сортировку массива данных и визуального
отображения процесса сортировки на экране. Первый поток производит сортировку по
возрастанию, второй по убыванию, третий производит графический вывод массива на
экран. Каждый поток работает с общим экземпляром массива данных. Состояние
сортировки выводится в одном элементе.
